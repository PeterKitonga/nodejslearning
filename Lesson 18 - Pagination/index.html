Adding Pagination

In this section, I learn how to paginate data and present pagination links dynamically. We paginate data so as to retrieve large data sets in chunks. This improves perfomance of the application and user experience

Pagination links

We first need a set of links that will help navigate between each chunk of data. We do this by passing query params to the route which will serve as pointers to the next 'page'. 

  1. <section class="pagination">
  2.   <a href="/?page=1">1</a>
  3.   <a href="/?page=2">2</a>
  4. </section>
  5.  
Pagination logic

For this we first need to set a variable that will determine how many items per page we want. This can be done as an ENV variable as well or be a dynamic value. We also use the .skip() function to serve as the offset of our chunked data and the .limit() function that will limit the amount of data we want to see per page. 

  1. // controllers/shop.js
  2. const Product = require('../models/product');
  3. const {
  4.   errorHandler
  5. } = require('../utils/error-handler');
  6.  
  7. const ITEMS_PER_PAGE = 3;
  8.  
  9. const getHome = (req, res, next) => {
  10.   const {
  11.     page
  12.   } = req.query;
  13.  
  14.   Product
  15.     .find()
  16.     .skip((page - 1) * ITEMS_PER_PAGE)
  17.     .limit(ITEMS_PER_PAGE)
  18.     .then(products => {
  19.       res.render('shop/index', {
  20.         prods: products,
  21.         page_title: 'Shop | Home',
  22.         route_name: 'home'
  23.       });
  24.     }).catch(err => errorHandler(err, next));
  25. };
Dynamic pagination logic and links

To make the pagination dynamic we first need to get the total number of documents in our query which we uses the .countDocuments() method. With this count we can calculate the previous, next, current and last pages which will help construct the pagination links. We also need to check whether there is a previous and next page. 

  1. // controllers/shop.js
  2. const Product = require('../models/product');
  3. const {
  4.   errorHandler
  5. } = require('../utils/error-handler');
  6.  
  7. const ITEMS_PER_PAGE = 2;
  8.  
  9. const getHome = (req, res, next) => {
  10.   let product_count = 0;
  11.   let {
  12.     page
  13.   } = req.query;
  14.   page = !page ? 1 : parseInt(page);
  15.  
  16.   Product
  17.     .find()
  18.     .countDocuments()
  19.     .then(document_count => {
  20.       product_count = document_count;
  21.  
  22.       return Product.find().skip((page - 1) * ITEMS_PER_PAGE).limit(ITEMS_PER_PAGE);
  23.     })
  24.     .then(products => {
  25.       res.render('shop/index', {
  26.         prods: products,
  27.         page_title: 'Shop | Home',
  28.         route_name: 'home',
  29.         pagination: {
  30.           total: product_count,
  31.           has_next_page: ITEMS_PER_PAGE * page < product_count,
  32.           has_previous_page: page > 1,
  33.           next: ITEMS_PER_PAGE * page < product_count ? page + 1 : null,
  34.           previous: page > 1 ? page - 1 : null,
  35.           current: page,
  36.           last: Math.ceil(product_count / ITEMS_PER_PAGE)
  37.         }
  38.       });
  39.     }).catch(err => errorHandler(err, next));
  40. };

With the logic in place we can construct which links are shown and hidden at a particular time. We can also determine which is the current page and add an active color to the respective link. 

  1. <section class="pagination">
  2.     <% if (pagination.current !== 1 && pagination.previous !== 1) { %>
  3.         <a href="/?page=1">1</a>
  4.     <% } %>
  5.  
  6.     <% if (pagination.has_previous_page) { %>
  7.         <a href="/?page=<%= pagination.previous %>"><%= pagination.previous %></a>
  8.     <% } %>
  9.  
  10.     <a href="/?page=<%= pagination.current %>" class="active"><%= pagination.current %></a>
  11.  
  12.     <% if (pagination.has_next_page) { %>
  13.         <a href="/?page=<%= pagination.next %>"><%= pagination.next %></a>
  14.     <% } %>
  15.  
  16.     <% if (pagination.last !== pagination.current && pagination.last !== pagination.next) { %>
  17.         <a href="/?page=<%= pagination.last %>"><%= pagination.last %></a>
  18.     <% } %>
  19. </section>
