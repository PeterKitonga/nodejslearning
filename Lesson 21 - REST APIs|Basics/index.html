Working with REST APIs - Basics

In this section, I refresh my knowledge on REST(Representational State Transfer) APIs. REST APIs are useful to provide access to data that is not rendered into HTML pages from a server-side application. They provide data to mobile apps, single page apps and even when accessing a 3rd party API like google maps.

Body Parser Middleware

Since we are no longer working with forms to post data we first need to setup the right body parser for POST, PUT and PATCH requests. We do this by adding a middleware that is available on express. This will allow parsing of application/json requests. 

  1. const express = require('express');
  2.  
  3. const app = express();
  4.  
  5. app.use(express.json());
Handling CORS

Because we will be accessing the API from a different origin, by default the browser rejects such requests as a security feature. To by pass this we can set some headers that allow requests from a specific or all origins. This is done using a middleware before any route is accessed. 

  1. const express = require('express');
  2.  
  3. const app = express();
  4.  
  5. app.use(express.json());
  6.  
  7. app.use((req, res, next) => {
  8.   res.setHeader('Access-Control-Allow-Origin', '*');
  9.   res.setHeader('Access-Control-Allow-Headers', 'Content-Type, Authorization');
  10.   res.setHeader('Access-Control-Allow-Methods', 'GET, POST, PUT, PATCH, DELETE');
  11.  
  12.   next();
  13. });
Returning JSON responses

JSON is a widely format used for REST API response and this can be implemented on Node.js using the res parameter in any route middleware. We use the .json() method available to pass data in JSON format and return a response. We can also configure the HTTTP status using the .status() method. 

  1. exports.getPosts = (req, res, next) => {
  2.   let data = [{
  3.     id: '1622030276720',
  4.     title: 'First Post',
  5.     content: 'This is the first post!'
  6.   }];
  7.  
  8.   res.status(200).json({
  9.     status: 'success',
  10.     data
  11.   });
  12. };
  13.  
  14. exports.addPost = (req, res, next) => {
  15.   const {
  16.     title,
  17.     content
  18.   } = req.body;
  19.  
  20.   res.status(201).json({
  21.     status: 'success',
  22.     data: {
  23.       id: Date.now().toString(),
  24.       title,
  25.       content
  26.     }
  27.   });
  28. };
Try it

To run the app, in your terminal run the following commands: 

  1. cd assets/scripts/lesson-21
  2.  
  3. npm install // if you have not installed dependencies
  4.  
  5. npm run serve // starts application
