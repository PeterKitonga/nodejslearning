SQL Introduction

In this section, I learned how to connect to an SQL(MySQL) database to store and retrieve data. This section briefly introduces setting up a connection to MySQL and the configurations passed.

Connection to a MySQL database

Before connecting to a MySQL database in Node.js we first need to install a dependency(MySQL client) that will give us the tools necessary to establish a connection and perform queries on a MySQL database. This dependency is installed via command npm install --save mysql2. We also need to configure a connection to the MySQL database created nodejs_learning. Below is how to create this connection through the client installed into the project. 

  1. const mysql = require('mysql2');
  2.  
  3. /**
  4.  * Connection pools keep the connection to the 
  5.  * MySQL server alive which means we can reuse 
  6.  * a previous connection. This improves latency
  7.  * of queries since we are not reconnecting every time.
  8. */
  9. const pool = mysql.createPool({
  10.     host: 'localhost',
  11.     user: [mysql username],
  12.     password: [mysql password],
  13.     database: 'nodejs_lessons'
  14. });
  15.  
  16. module.exports = pool.promise();
Querying a MySQL database table

We can now perform a query after establishing a connection to the database. This can be done using the .execute() promise which allows us to write raw SQL queries to fetch, create, update and delete data. Below is an example query used to fetch all products in our Product model. 

  1. // imports database pool connection configuration 
  2. const database = require('../utils/database');
  3.  
  4. class Product {
  5.     constructor({ id, title, image_url, description, price }) {
  6.         this.id = id;
  7.         this.title = title;
  8.         this.image_url = image_url;
  9.         this.description = description;
  10.         this.price = price;
  11.     }
  12.  
  13.     static fetchAll() {
  14.         // returns a promise with results from raw SQL query
  15.         return database.execute('SELECT * FROM products');
  16.     }
  17.  
  18.     ...
  19. }
  20.  
  21. module.exports = Product;
Inserting data into a MySQL database table

We can also use the .execute() promise to add data to our table. Below is a snippet where we are saving a new product. 

  1. const database = require('../utils/database');
  2.  
  3. class Product {
  4.     constructor({ id, title, image_url, description, price }) {
  5.         this.id = id;
  6.         this.title = title;
  7.         this.image_url = image_url;
  8.         this.description = description;
  9.         this.price = price;
  10.     }
  11.  
  12.     ...
  13.  
  14.     save() {
  15.         /**
  16.          * Inserts data where it assigns values in the order in which they are defined
  17.          * corresponding to the order of fields defined in the query
  18.         */
  19.         return database.execute(
  20.             'INSERT INTO products (title, price, image_url, description) VALUES (?, ?, ?, ?)',
  21.             [this.title, this.price, this.image_url, this.description]
  22.         );
  23.     }
  24.  
  25.     ...
  26. }
  27.  
  28. module.exports = Product;
Updating data of a MySQL database table

Below is a snippet where we are updating an existing product. 

  1. const database = require('../utils/database');
  2.  
  3. class Product {
  4.     constructor({ id, title, image_url, description, price }) {
  5.         this.id = id;
  6.         this.title = title;
  7.         this.image_url = image_url;
  8.         this.description = description;
  9.         this.price = price;
  10.     }
  11.  
  12.     ...
  13.  
  14.     update() {
  15.         return database.execute(
  16.             'UPDATE products SET title = ?, price = ?, image_url = ?, description = ? WHERE id = ?',
  17.             [this.title, this.price, this.image_url, this.description, this.id]
  18.         );
  19.     }
  20.  
  21.     ...
  22. }
  23.  
  24. module.exports = Product;
Deleting data from a MySQL database table

Below is a snippet where we are deleting an existing product. 

  1. const database = require('../utils/database');
  2.  
  3. class Product {
  4.     constructor({ id, title, image_url, description, price }) {
  5.         this.id = id;
  6.         this.title = title;
  7.         this.image_url = image_url;
  8.         this.description = description;
  9.         this.price = price;
  10.     }
  11.
