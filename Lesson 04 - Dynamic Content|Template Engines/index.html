Dynamic Content & Template Engines

In this section, I learned about template engines and using them to display dynamic data. There are number of templating engines available and in this section I learned how to use pug, ejs and handlebars. To use templating engine we first need to install them via npm e.g npm install --save pug. Afterwards we need to let express know which template engine we are going to use via the set method. 

  1. const express = require('express')
  2. const app = express()
  3.  
  4. app.set('view engine', 'pug') // tells express which template engine is in use
  5. app.set('views', 'views') // tells express where our template files

For handlebars we require to pass the imported library to the app.engine middleware. This is because handlebars is not automatically loaded by nodejs like pug. 

  1. const express = require('express')
  2. const expressHbs = require('express-handlebars')
  3.  
  4. const app = express()
  5.  
  6. // with handlebars, it is not automatically loaded like pug. We need to pass it to the engine method
  7. app.engine('hbs', expressHbs({layoutsDir: 'views/layouts/', defaultLayout: 'wrapper', extname: 'hbs'}))
  8. app.set('view engine', 'hbs') // tells express which template engine is in use
  9. app.set('views', 'views') // tells express where our template files
Try it

To run the app, in your terminal run the following commands: 

  1. cd assets/scripts/lesson-04
  2.  
  3. npm install // if you have not installed dependencies
  4.  
  5. npm run serve // starts application
